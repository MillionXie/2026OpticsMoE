# Pretrained Backbones

Place frozen source OpticalMoE backbones here before running transfer adaptation.

The default transfer configs look for a dataset-switching OpticalMoE trained on:

- `mnist`
- `fashionmnist`
- `emnist_letters`

Default expected directory:

`opticalmoe_experiments/transfer_adaptation/pretrained_backbones/dataset_switching_moe_mnist_fashion_emnist_letters/`
